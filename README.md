# Calculator

A test calculator built for testing the **Lumen Reviewer**.

## Purpose

This project serves as a simple calculator application used as a test subject to evaluate and review the Lumen Reviewer tool. It is not intended for production use.

## Usage

This repository is used internally for testing and reviewing purposes only.

Supported operations:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulo (`%`)

## Notes

- This is a test project for Lumen Reviewer evaluation.
